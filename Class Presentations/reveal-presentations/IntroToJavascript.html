Conjunction Junction What’s a Function?

## Function * Organized lines of code that complete a job * Is executed when invoked (such as clicking on a button) Citation:http://www.w3schools.com/js/js_functions.asp
## Called by Any Other Name, It Doesn’t Work * Functions must be named * Functions must be called by their name * Functions need the keyword “function” to be identified as a function * Functions end in a parenthesis () which may include parameters
## (string) * Parameters define what must be passed into a function for it to function * Functions can have more than one parameter * Parameters are separated by commas , * Once passed inside it can be used by the function as a local variable
## But what’s a variable?? I don’t know… you tell me.
## var In javascript you define a variable with the keyword var Variables can be strings (think text), numbers (with or without decimals) or booleans (true or false) ```javascript var string = “This is a string and it uses quotation marks”; var number = 3; var boolean = true; ```

## It’s like magic Functions need to be invoked by: * a user action (mouse roll or click) * the browser event (page loading) * being called by another function * or calling itself
## Functions are Giving * They can return values * The code inside the function stops executing when it hits a return statement * You must use the return keyword to define what is being provided from the function
