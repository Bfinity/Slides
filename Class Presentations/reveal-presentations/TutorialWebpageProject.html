What is an event handler? (What does it do?)

An event handler is a structure in HTML or JavaScript that allows an element to respond to user interaction with an HTML element.

How does the inline (first) style of event registration work?

The response is written in as a value of a property on an HTML element.

How does the traditional (second) style of event registration work?

The response is written in JavaScript and it references the HTML element.

How does the W3C (third) style of event registration work? What is an important difference between W3C style and traditional style?

The response is written in JavaScript and it references the HTML element. It has a different syntax from the second method, and it allows the user to determine which element the browser responds to first: the parent (event capturing) or the child (event bubbling).

What is an array?

An array is an object that stores many values in a single variable.

What kinds of objects can arrays hold?

Arrays can hold booleans, numbers, strings, and other objects. (Really, these are all objects.)

Could this be an array?

["ten", 9, 8, "7", 6]

Yes, because JavaScript arrays can hold different data types (all are objects).

How would you get a reference to "7"?

var numbers = ["ten", 9, 8, "7", 6]
