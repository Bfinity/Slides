What is a control structure? (Feel free to use your own words.)

"A control structure is a block of programming that analyzes variables and chooses a direction in which to go based on given parameters." -Wikiversity

What is an if statement?

A control structure that checks if a given boolean variable or expression is true

How would you write an if statement that increments i if i is less than 6?

```javascript if(i < 6) { i++; } ```

How is an if-else statement different from an if statement?

An if-else statement provides an instruction for if the condition is false

How would you write a statement that increments i if i is less than 6, but decrements it otherwise?

```javascript if(i < 6) { i++; } else { i--; } ```

What is a switch?

A switch is a control structure that allows the value of a variable to change the flow of execution

What does a break statement do in a switch statement?

It prevents the statements in the switch block from "falling through" (all just being executed in sequence)

What is the default label in a switch statement for?

The default label creates an option for when the variable does not match any of the cases.

Write a switch statement for a traffic light.

(Example) ```javascript var color = "green"; switch (color) { case "green": speed = 35; break; case "yellow": speed = 45; break; case "red": speed = 0; break; default: speed = 10; } ```

What is a for loop?

A for loop is a control structure that allows a statement to be repeated until a certain condition is met.

Let's write a for-loop that tells someone to clap ten times.

(Example) ```javascript for(var i=0; i<10; i++) { clap(); } ```

What is a while loop?

A while loop is a control structure that allows a statement to be repeated while a certain condition is under effect. The condition is checked at the beginning of each loop.

Let's write a while-loop that tells someone to clap ten times.

(Example) ```javascript var i = 0; while(i<10) { clap(); i++; } ```

What is a do-while loop? (How does it differ from a while loop?)

A do-while loop is a control structure that allows a statement to be repeated while a certain condition is under effect. However, the code within a do-while looop is always executed at least once, and the condition is checked at the end of each loop.

Let's write a do-while loop that tells someone to clap ten times.

(Example) ```javascript var i = 0; do { clap(); i++; } while (i < 11); ``` or ```javascript var i = 1; do { clap(); i++; } while (i <= 10); ``` (many different ways to do this)

Switch

```javascript var color = "green"; switch (color) { case "green": speed = 35; break; case "yellow": speed = 45; break; case "red": speed = 0; break; default: speed = 10; } ```
## Switch * Allows for multiple options in cleaner way than if-else-else-else-... * Condition would normally not be defined right above the switch (that's not too imaginative)
## What about break; and default: ? * break; stops execution of the code in the blocks * The last case (or default, if it's last) do not need to end with break; * default: is used when the condition does not match the switch statement cases

## Loop-de-loop

For Loop

```javascript for (i=0; i<6; i++) { changeText("This is time number " + i) } ```
## For Loop * initialization: the counter variable is initialized at this value. ```javascript for (i=0; i<6; i++) { changeText("This is time number " + i) } ```
## For Loop * condition: the for-loop keeps running until this becomes false. ```javascript for (i=0; i<6; i++) { changeText("This is time number " + i); } ```

A Note on Infinite Loops

  • They occur in loops when the condition is never false.
  • They lead to the code running and running until your browser runs out of memory, which will break your script.
  • They are very bad. Avoid them by checking your conditions carefully.
## For Loop * update: After each loop, the counter variable is changed in this way. * "++" means incremented by 1. For example, if i was 2, it becomes 3. ```javascript for (i=0; i<6; i++) { changeText("This is time number " + i) } ```

While Loop

## While Loop * Great way to keep executing a block of code until something changes. * Make sure that the condition can eventually change, though * Increase the counter variable within the block (i++)
