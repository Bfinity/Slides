## But first… More Vocabulary Words!
## Operator * An operator is a symbol used to assign or compare values and complete arithmetic.
## The Assignment Operator = * is used to assign a value to a variable. Example: var three = 3; This assigns the value of 3 to the variable three.
## Math Operators We’ll start with familiar faces...
`+` : Addition - is used to add variable values together.
`-` : Subtraction - is used to subtract variable values from one another.
`*`: Multiplication - is used to multiply variable values together
`/` : Division - is used to divide variable values from one another.
## Math Operators Cont. Now for some new friends…
% : Modulus - is used to get the reminder in a division
++ : Increment - is used to count up by one
-- : Decrement - is used to count down by one
Examples:
var one = 5 % 2; * This assigns 5 modulus 2, which is 1 to variable one.
var five = ++4; * This assigns the increment of 4, which is 5 to variable five.
var three = --4; * This assigns the decrement of 4, which is 3 to variable three.

Cite: http://www.w3schools.com/jsref/jsref_operators.asp

## If, Else, Else If **if** is used to evaluate if a condition is true. If it is then the code inside it will be executed.
**else if** is used to add another conditional to be evaluated for true. If it is true then the code inside will be executed.
**else** is used to execute code when the previous if condition is false. If it is false then the code inside will be executed. http://www.w3schools.com/jsref/jsref_if.asp

## Switch Sometimes you have to evaluate a number of conditions for one function. In times like those, instead of if..else if… else if… else if… else, you can use a switch statement.

**switch** evaluates if one or more multiple conditions are true. If so, the code inside the block will be executed.
## What is this function doing? ```javascript Function favoriteColor(color){ switch(color){ case “blue”: document.body.style.backgroundColor = “blue”; break; //this stops the next case from being evaluated. case “red”: document.body.style.backgroundColor = “red”; Break; case “yellow”: document.body.style.backgroundColor = “yellow”; break; } }``` When color = blue, what happens?
