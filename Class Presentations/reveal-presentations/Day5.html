What is HTML?

(HyperText Markup Language; markup language used to create websites)

What is a block element?

(element that starts on a new line; examples include headers, p, lists and list items)

What is an inline element?

(element that continues on the same line; examples are spans, images, and semantic markup like em or b)

What is a div used for?

(grouping block elements/a block of elements together in their own block element)

What is a span used for?

(grouping inline elements together, often for styling)

What does CSS stand for? What is it used for?

(cascading style sheets, changes presentation of a webpage)

What is this an example of?

```css .file { background-color="#000000"; } ```

(a CSS rule)

What are the two parts of a CSS rule?

(selector and declaration)

What are the two parts of a declaration?

(property and value)
