Attributes

  • Key/Value Pairs
  • Keys define the attribute
  • Values set the value of the attribute
  • Provides additional info about/Modifies HTML elements
  • Always specified in the start tag
<h1 align=“center”>

Yes you can! Using the dir attribute you can change the direction of the text in an element. There are about 118 different HTML tags though not all are supported by HTML5. And these tags can be modified by many different attributes.

<a> </a>

  • Defines a hyperlink
  • Needs href attribute to define link destination
  • By default,
    • Unvisited links are in blue underlined text
    • Visited links are purple underlined text
    • Active links are red underlined text
<a href=“http://www.detroitedisonpsa.org”>DEPSA</a>
